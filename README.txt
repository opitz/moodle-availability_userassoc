Moodle 4.5 availability condition plugin: availability_userassoc

Adds a Restrict access condition "User association" that checks the first character of the custom user profile field with shortname employee_details.

Use:
- On an activity (e.g. LTI), add Restrict access -> User association
- Allowed letters: e.g. S,U,P,V,H
- Alumnus and casual staff will not have that field populated

Letters meaning in the UCL context:
S-staff
U-undergraduate student
P-postgraduate student
V-visitor
H-honorary staff
